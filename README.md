# ELITRIX EXPERTS — site multipage redesign

Prototype HTML/CSS/JS fonctionnel basé sur le fichier HTML fourni de la page actuelle ELITRIX EXPERTS.

## Pages incluses
- `index.html` — accueil
- `a-propos.html`
- `votre-projet.html`
- `secteurs.html`
- `services.html`
- `simulateurs.html` — simulateurs interactifs
- `solutions-digitales.html`
- `blog.html` — filtres d’articles
- `carrieres.html`
- `contact.html` — formulaire prototype + mailto
- `mentions-legales.html`
- `politique-confidentialite.html`

## Images
Les 7 images détectées dans la page source sont réutilisées via leurs URLs originales `elitrix-experts.fr`. Le fichier `assets/data/original-images.json` liste les sources.

## Fonctionnalités
- navigation multipage
- menu mobile
- formulaire de contact avec validation + mailto + stockage local de démonstration
- simulateurs cotisations / impôt / indemnité kilométrique
- filtres du blog
- FAQ accordéon
- bannière cookies prototype
- responsive design

## Mise en production
Pour un vrai site WordPress/IONOS, importer les images localement dans la médiathèque, remplacer les URLs d’images si besoin, connecter le formulaire à un outil réel et faire valider les simulateurs par le cabinet.
